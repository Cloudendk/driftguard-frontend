Velkommen

DriftGuard bliver din kundeportal til sikkerhedsstatus, drift-detektion og anbefalinger på tværs af tenants.

Denne side er en “baseline landing page”, så deployment pipeline og domæner kan valideres først. Næste step bliver login (Entra ID) og tenant-vælgning (fx lindemose.com).

Log ind (kommer snart) GitHub repo SWA default hostname

Det her validerer vi nu

  • CI/CD: GitHub Actions deployer en rigtig side
  • Routing: `/login` route findes (placeholder)
  • Domæne: senere `clouden.dk/driftguard` via host-site routing
  • Security baseline: headers + auth i næste iteration

Når dette deployer “grønt”, bygger vi React/Vite UI + Entra auth på en kontrolleret måde.
